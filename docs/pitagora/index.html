しかけを使ってナゾを解く
"永久機関"のピタゴラパズル

「ピタゴラスの永久機関」は、しかけを使って"永久機関"を作っていく"ピタゴラパズルゲーム"です。

転がる岩を曲げたり、止めたり、打ち出したり、重しにしたり…
しかけの種類は様々で、時には自分自ら丸まって岩と一緒に転がって、塔に秘められた"永久機関"のナゾを解いていきます。

たどり着いたのは天を突く巨大な塔。
塔の力の正体は？主人公の目的は？塔の頂点には何が待つのか？
残された石碑・パピルスを集めれば何かが分かるかもしれません。

40以上のステージで遊びごたえも十分。
どうしても解けない時はヒントやスキップがあるので、パズルが苦手なひとでも安心です。
