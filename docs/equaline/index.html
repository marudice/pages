一筆書きでターゲットを作り出せ。

一筆で線を描いてターゲットを計算。

「通った順番で計算される」。
たった1つのルールと9個のマスが脳に挑戦状を叩きつける。
そこに運など、必要はない。

3つのエンドレスモードで限界に挑戦。

エンドレスモードは問題の難易度に応じて分けられている。
もし詰まった時は、残り時間を対価に問題を変えることもできる。　
その挑戦に、終わりはない。

特殊なルールのミッションや実績を制覇しろ。

制限時間3秒・ターゲット不明、といった特殊なミッションが全24種類。
さらに、ミッションや実績の解禁条件は、式の長さ・残り時間など多岐に渡る。
EQUALINEの全てを、遊びつくせ。
