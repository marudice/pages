ひらめきのナゾと論理のパズル
小さな記憶の物語

気が付くと樹上の家に居た”私”
出口はどこ?
あの女の子とおじいさんは誰?
私がここにいる理由は?

「Treehouse Riddle」は、ナゾやパズルを解いて物語を進めていく
”ナゾｘパズル　アドベンチャー”です。

樹上の家の一室に閉じ込められていた主人公。
なつかしいドット絵で描かれた部屋を探索、
ナゾとパズルを解いて脱出の手がかりを探ります。

家と同じく温かい木で描かれたナゾやパズル。
30以上の知恵の挑戦をじっくりお楽しみ頂けます。

全てのナゾでヒントとスキップが使用可能。
ナゾ解きが苦手な人でも詰む心配はありません。
